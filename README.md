## Description
A 3D WebGL game where the player explores a world and collects treasures.

## Controls
- WASD -> Move
- E -> Collect items

## Features
- WebGL rendering pipeline
- Phong lighting (ambient, diffuse, specular)
- Textured objects
- Collision detection
- Interactive gameplay

## Textures Folder
This folder contains all image files used as textures in the WebGL scene.

Textures are applied to 3D objects to give them realistic appearance instead of plain colors.

### What goes here
Image files such as:
- .jpg
- .png
